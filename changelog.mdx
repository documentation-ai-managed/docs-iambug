## Recent Updates

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Introduce real-time collaboration on documentation pages, allowing multiple users to edit simultaneously.
- Add AI-powered content suggestions to help you draft sections faster.

## Improvements

- Enhance search functionality with fuzzy matching and full-text indexing for better results across your docs.
- Optimize page load times by `>30%` through improved asset bundling.

## Bug Fixes

- Fix intermittent sync issues when editing large documents.
- Resolve authentication errors during webhook deliveries.

</Update>

## Upgrade Instructions

Stay current with iambug by following these steps. You receive automatic notifications in your dashboard at `https://dashboard.example.com`.

<Callout kind="info">
  Always back up your projects before upgrading to avoid data loss.
</Callout>

<Steps>
  <Step title="Check Compatibility" icon="check-circle">

    Review the release notes above for breaking changes.

  </Step>
  <Step title="Update Client SDK" icon="download">

    Use your preferred package manager to get the latest version.

    <CodeGroup tabs="npm,yarn">
      ````bash
      npm install iambug@latest
      ````

      ````bash
      yarn add iambug@latest
      ````
    </CodeGroup>

  </Step>
  <Step title="Run Migration" icon="settings">

    Execute the migration script if required for your version.

    ````bash
    npx iambug migrate --project your-project-id
    ````

  </Step>
  <Step title="Verify" icon="zap">

    Test key features like editing and sharing in a staging environment.

  </Step>
</Steps>
