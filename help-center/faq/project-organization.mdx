## Overview

iambug lets you organize projects into "docspaces."

## Using Docspaces

1. Create a new docspace for each project
2. Add folders to group topics
3. Use tags to label and filter docs
4. Collaborate by inviting team members

## Tip
Organize docs by feature or workflow for easy navigation.
